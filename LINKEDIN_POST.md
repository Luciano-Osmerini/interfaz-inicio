# Post sugerido para LinkedIn

## Version corta (lista para publicar)
Construi una solucion web interna con Flask + SQL Server para centralizar acceso por rol, gestionar usuarios y dar trazabilidad a procesos operativos.

La aplicacion incluye:
- autenticacion segura con sesiones y hash de password
- alta de cuentas y cambio de clave
- gestion de usuarios por superadmin
- bitacora de acciones filtrable
- carga/descarga de archivos de procesos

Este proyecto me permitio mejorar en arquitectura backend, seguridad y diseño de flujos internos.

Si te interesa, comparto demo y detalles tecnicos. Feedback bienvenido.

## Version con enfoque en impacto
En este proyecto desarrolle una web app para ordenar operaciones internas y reducir pasos manuales en procesos de gestion.

Resultados logrados:
